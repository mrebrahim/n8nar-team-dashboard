# n8nar Team Dashboard

لوحة متابعة فريق n8nar.com — مبنية بـ React + Supabase

## الميزات
- تسجيل دخول لكل موظف
- تتبع المهام اليومية
- رفع التقارير اليومية
- لوحة المدير: KPIs + السوشيال ميديا + تقارير الفريق

## التقنيات
- React + Vite
- Supabase (قاعدة البيانات)
- Windsor.ai (بيانات السوشيال)
- Vercel (الاستضافة)
